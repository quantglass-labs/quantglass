# Code Of Conduct

QuantGlass is a technical project. Discussion should stay direct, respectful,
and focused on improving the software.

## Expected Behavior

- Be respectful and professional.
- Assume good intent, but ask for evidence when technical claims matter.
- Critique code, data, tests, and designs rather than people.
- Make risk, security, and financial-disclaimer concerns explicit.
- Help newcomers find the right docs or issues.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or conduct.
- Publishing private information.
- Spam, scams, market manipulation, pump-and-dump promotion, or financial advice
  presented as project guidance.
- Intentionally submitting malicious code, secrets, or misleading market data.

## Enforcement

Maintainers may remove comments, close issues, block users, or reject
contributions that violate this code of conduct.

Report serious conduct issues privately to the maintainers.
