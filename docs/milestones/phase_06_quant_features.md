# Phase 06 - Quant Features

## Goal

Replace handcrafted chart overlays and indicator values with backend-generated features.

## Checklist

- [ ] Compute EMA and SMA features.
- [ ] Compute RSI and MACD features.
- [ ] Compute Bollinger, ATR, and volume features.
- [ ] Compute support/resistance and regime signals.
- [ ] Expose feature payloads through contracts and backend APIs.

## Acceptance Gates

- Symbol detail indicator panels and overlays are backend-driven.
- Feature computation is reproducible from normalized data.
